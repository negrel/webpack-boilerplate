# Webpack - boilerplate

_Webpack boilerplate with babel, eslint (airbnb config), prettier, SASS/SCSS support._

If dependencies are out of date :

```bash
# Install ncu cli tool
npm install -D ncu

# Update all your dependencies
npx ncu -u

# Install your up to date dependencies
npm install
```
